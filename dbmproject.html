Project Overview

My group's project for our database management course focused on creating an easily accessible and sophisticated database for analyzing team performances in the 2014 NCAA Division 1 basketball season. Leveraging data from over 11,495 games, we refined a large dataset from Kaggle to provide in-depth statistical insights through a web application hosted on Oracle APEX.

Data Refinement

We streamlined the dataset to focus on pertinent statistics by reducing over 150 data columns down to 40. Our goal was to present this data effectively through a web-based application, optimizing for user experience and analytical value.

Database Implementation and Analysis

Utilizing SQL within Oracle APEX, we constructed a relational schema comprising four interconnected tables. Our analysis delved into performance metrics like field goal percentages and game outcomes under various conditions, such as neutral audiences and tournament seeds.

Results and Web Design

The project culminated in insightful findings on team performances, particularly under winning and losing conditions, and their correlation with three-point shots. The web application we designed provides users with an interactive and comprehensive exploration of the NCAA Tournament statistics.
