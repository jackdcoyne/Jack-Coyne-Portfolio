Project Overview

Building this site for my digital product management class was an exercise in applying agile principles and user-centered design to create a dynamic and engaging web experience. Starting with a Bootstrap template, the project evolved over a series of focused development phases.

Agile Development & Iterative Enhancement

The website's creation process reflected the essence of agility, with each phase introducing strategic functionalities. This iterative approach ensured that every aspect of the website was refined and aligned with the goal of providing a seamless user experience.

Key Components

A pivotal component of the development was leveraging Google Analytics to gather insights into user behavior and traffic patterns. This powerful tool provided the data necessary to make informed decisions about design and functionality enhancements.

Learning and Personal Growth

This project has been an enriching experience, blending the lines between classroom theory and real-world application. Each phase of development has brought its own set of challenges and triumphs, significantly enhancing my expertise in digital product management.

Other Product Management Endeavors

In my Digital Product Management course, I also worked on an AI Salary Predictor, a dynamic web application that employs a machine learning model to predict salaries based on user input. The app's frontend API allow users to enter various parameters, such as age, gender, country, and more, to receive a personalized salary prediction. Check out the project if you're interested: AI Salary Predictor on Azure, AI Salary Predictor on GitHub.
